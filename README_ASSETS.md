# Roar Arena Final Website Assets

Drop these into your Next.js `public/` folder.

## Logos
Use `public/logos/logo-lockup-dark-transparent.png` for dark theme header/footer.
Use `public/logos/logo-lockup-light-transparent.png` for light theme header/footer.
Use `public/logos/logo-icon-dark-transparent.png` and `logo-icon-light-transparent.png` for favicons, small badges, and mobile icon marks.

## Backgrounds
Use `public/backgrounds/hero-bg-dark.png` and `hero-bg-light.png` as subtle hero background overlays.

## Posts
Use post images from `public/posts/` inside the Latest From Arena grid and the View Post modal.

## OG
Use `public/og/og-image.png` for social share image.
